# UI/UX

`UX(User Experience)` - 사용자 경험

`UI(User Interface)` - 사용자 상호 작용

디지털 기기, 사용자, 상호 작용

### 현업에서의 UI/UX

회사 규모에 따라 역할이 달라진다.

사용자 - 기획자

상호작용 - 디자이너

디지털 기기 - 개발자

### 개발자 관점의 현업

기획자 → 디자이너 → 개발자

### 유능한 개발자

✔️ 가능/불가능한 기능 및 예외 사항에 대한 인식

✔️ 이를 정확하게 납득시킬 수 있는 설득력

✔️ 그 기능을 요구하게 된 기획자/디자이너 의도 파악

✔️ 최대한 요구와 일정에 맞출 수 있는 개발 실력

### 개발할 때 고려할 점

- 웹/모바일
- 스펙에 없는 기능
